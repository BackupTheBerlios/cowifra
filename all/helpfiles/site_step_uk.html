The wizard step navigation
At the top of the main step site window you find the quick step navigation. If you don't want to change something in the actual step or if you like to review the last step, you can use this navigation. The "Go to wizard index" link will always bring you back to the wizard start page. Next and previous step link are only available if there is a next or previous link.
 
Step dependencies
If this section apears close to the main window header, there are dependencies with other steps. That means, if you don't use the wizard step by step, you should at least take a look at the listed steps.
 
The step configuration
The rest of the main window is the step configuration. Normally you will find first and description of the step and after that an input field. In most cases there ist a selection box too, which let you choose between the active, the standard and further comment values. If you change the value of the step, the configuration wizard framework tries to save the old value as a comment value, so you can restore the value easily by select them in the selection box.
