Wizard files (*.cwf)
Each wizard for a application is saved in one wizard file, stored in the "wizards" subdirectory. The wizard information are stored in the xml-format, released by the w3c. The rules for a xml wizard files are defined in the document type definition (dtd) file called wizard.dtd and is locacted also in the wizards subdirectory.

If you like to configure an linux/unix application with the configuration wizard framework (cwf) you need the right wizard file for that application. If you don't own the right wizard file you have to download it from the web. Keep attention of the right application version! On the index site of the cwf you find a list of all available wizards on your system. If the right wizard for your application is not available get it from http://cwf.berlios.de (adress not available at the moment).
