The Mainmenue
This is the mainmenue like you will see it, on every site of the configuration wizard framework (cwf). If you don't want to use this menue you can hide it, by using the small button right before the "CWF Mainmenue" string. The meaning of each button is described next:
Home
Return to the configuration wizard framework start page. On this site you can choose between all alvailable wizards.
Help
Will always give you informations about buttons and what to do one each site.
Beginner design
With this and the next two buttons you can select a special design. Beginner design means bigs icons and text for each option.
Advanced design
Advanced design only uses small icons and text.
Professional design
This design uses no icons, it only displays the text as links.
Available language
If a flag button appears in several colors you can choose this language.
 
 
The wizard menue
This is the wizard selection window. Each wizard that is available in your system is listed in this box. You can start a wizard just by clicking on the icon or the name of the wizard. If no wizards is available, you have to get wizards from the word wide web. Please take a look at the help topic "Wizard files".

You can hide the content of this window by clicking the small button in the headline of this window. For unhiding do the same again.
