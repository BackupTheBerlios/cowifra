Das Hauptmenü
Dies ist das Hauptmenü. Es befindet sich im oberen Teil jeder Seite des Konfigurations Wiarzard Framework (CWF). Sie können das Menü mit Hilfe des kleinen runden Buttons in der Kopfzeile aus- und wieder einschalten. Die Bedeutung der einzelnen Schaltflächen folgt:
Startseite
Mit diesem Link gelangen Sie jederzeit zuück auf die Startseite des CWF. Auf dieser Seite können Sie einen Wizard aus den vorhandenen Wizards auswählen.
Hilfe
Mit Hilfe dieses Links können Sie jederzeit eine neue Seite mit aktuellen Hilfe Informationen zur aktuellen Seite öffnen.
Anfänger Design
Die Göße und Anzeige aller Buttons lässt sich über diesen und die folgenden zwei Buttons verändern. Im Design "Anfänger" werden alle Icons groß und ein entsprechender Text angezeigt.
Fortgeschrittenes Design
In diesem Design erscheinen werden nur kleine Icons und entsprechende Texte verwendet.
Profi Design
In diesem Design werden keine Icons verwendet. Nur die Texte sind als Links nutzbar.
Verfügbare Sprache
Bunte Flaggenicons bedeuten, dass die entsprechende Sprache zur Verfügung steht und ausgewählt werden.
 
 
Das Wizard Menü
Dieses Fenster zeigt alle verfügbaren Wizards in Ihrem System an. Um einen Wizard zu benutzen, klicken Sie einfach auf den entsprechenden Namen. Sollte hier kein Wizard angezeigt werden, müssen Sie zunächst entsprechende Wizard Dateien aus dem World Wide Web (WWW) beschaffen. Nähere Informationen dazu bietet das Hilfethema "Wizard Dateien".

Sie können das Fenster mit Hilfe des runden kleinen Buttons in der Titelleiste des Fenster aus- und wieder einblenden.
