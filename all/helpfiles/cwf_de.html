Das Konfigurations Wizard Framework (CWF)
Das Konfiguratins Wizard Framework wurde entwickelt um Linux bzw. Unix Anwendungen einfach in einer Schritt für Schritt Prozedur (Wizard) zu konfigurieren. Es ist leicht mit Hilfe von sogenannten Wizard Dateien erweiterbar, die auf dem XML-Standard basieren.

Für jede Applikation die mit dem CWF konfiguriert werden soll wird eine passende Wizard Datei benötigt. Wenn diese Datei existiert, wird der entsprechende Wizard auf der CWF Homepage angezeigt und kann ausgewählt werden. Mehr Informationen zu den Wizard Dateien und Bezugsquellen finden Sie unter dem Hilfe Punkt "Wizard Dateien"

Möchten Sie eigene Wizard Dateien entwickeln, sollten Sie einen Blick in die Anleitung werfen.

Das CWF wurden in Zusammenarbeit mit der FOKUS-Fraunhofer / Berlios.de / Lutz Henkel von Christian Schmidt (smt@fokus.fraunhofer.de) als Diplomarbeit 2003 geschrieben.
