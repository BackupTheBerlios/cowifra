Die Wizard Schritt Navigation
Als erstes finden Sie in jedem Wizard Schritt die Schritt Navigation. Wenn Sie einmal in einem Schritt keine Änderung durchführen möchten, können Sie mit Hilfe dieser Navigation bequem zum nächsten Schritt gelangen. Mit Hilfe des Links "Zur Wizard Übersichtsseite" können Sie jederzeit zur Wizard Startseite gelangen.
 
Schritt Abhängigkeiten
Existieren zu einem Schritt, noch weitere Schritte, die in enger Verbindung zu dem aktuellen Schritt stehen, so erscheint direkt unter der Schritt Navigation der Abschnitt "Abhängigkeiten". Dieser zeigt Ihnen alle Schritte in denen evt. ebenfalls Änderungen durchgeführt werden sollten, wenn Sie die Einstellung der aktuellen Option ändern.
 
Die Schritt Konfiguration
Der Rest des Hauptfensters ist der Konfiguration der aktuellen Option vorbehalten. Er beginnt in der Regel mit einer ausführlichen Beschreibung der aktuellen Option, gefolgt von einer Eingabemöglichkeit (Input Feld, Select Box). In vielen Fällen ist eine weitere Auswahlbox vorhanden, die Ihnen zum einen deaktivierte Werte Ihrer Konfiguration anzeigt und zum anderen den default Wert für die aktuelle Option. Sollten Sie den Wert einer Einstellung ändern, so wird dieser (falls möglich) als auskommentierter Wert in der aktuellen Konfigurationsdatei gespeichert und kann beim nächsten mal über die Auswahlbox reaktiviert werden.
