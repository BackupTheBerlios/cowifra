Die Wizard Schritt Auswahl
Dieses Fenster zeigt alle Schritte an, die in diesem Wizard zu Verfügung stehen. In der Regel wird ein Wizard genutzt, in dem der "Start-Button" geklickt wird, anschließend werden automatisch allen nötigen Schritte durchlaufen. In manchen Fällen ist es jedoch möglich, dass Sie nur eine Option in der Konfiguration zur Ihrer Applikation mit Hilfe des CWF ändern möchten. In diesem Fall können Sie einfach den entsprechenden Schritt direkt auswählen. Der Schritt, dessen Icon einen roten Haken besitzt, ist dabei stets der Schritt, der aktuell angezeigt bzw. bearbeitet wird. Auch dieses Fenster können Sie bei bedarf mit Hilfe des kleinen runden Buttons in der Kopfzeile ein- oder ausblenden lassen.
 
Das Aktionen Fenster
Bei einigen Konfigurationen ist es notwendig nach einer Änderung bestimmte Programme zu starten, damit die gemachten Änderungen auch umgesetzt werden. Wenn dieses Fenster in einem Wizard vorhanden ist, zeigt es alle Aktionen an, die zum Abschluß einer Konfiguration durchgeführt werden sollten. Um dies zu vereinfachen befindet sich zu oberst der Button "Alle Aktionen ausführen". Sollten Sie die Auswirkungen der einzelnen Aktionen genau kennen, können Sie diese auch einzeln ausführen.

Auch dieses Fenster können Sie bei bedarf mit Hilfe des kleinen runden Buttons in der Kopfzeile ein- oder ausblenden lassen.
 
Hauptfenster: Kopfbereich
Der erste Abschnitt des Hauptfenster zeigt Ihnen verschiedene Informationen über den von Ihnen ausgewählten Wizard. An dieser Stelle sollten Sie sich davon überzeugen, dass Ihre Applikation die gleiche Version besitzt, wie die Applikation für die der Wizard erstellt wurde. Sollten es ein Problem mit dem aktuellen Wizard geben, finden Sie hier ebenfalls die E-Mail Adresse des Autors.
 
Hauptfenster: Wizard Start Button
In der Regel wird ein Wizard über diesen Button gestartet und anschließend werden alle Schritte nacheinander durchlaufen. Die Zahl in Klammern gibt dabei die Anzahl aller Schritte an, in diesem Beispiel also 44.
 
Hauptfenster: Wizard Gruppierungen (Thema und User Level)
Ob diese Möglichkeiten zur Auswahl stehen hängt vor allem vom Entwickler des Wizard Files ab, Sie müssen nicht zwingend vorhanden sein. Es gibt zwei verschieden Gruppierungsarten. Zum einen die Gruppierung nach den User Fähigkeiten, bei der Sie selbst entscheiden müssen, welche Fähigkeiten Sie besitzen und welche Optionen Sie somit konfigurieren möchten. Zum anderen die Gruppierung nach Themen. Wenn Sie also nur einen Teil der Konfiguration ändern möchten, wählen Sie hier das entsprechende Thema, soweit vorhanden. Im oben gezeigten Beispiel sind zwei verschiedene Benutzerlevel und drei verschiedene Themen definiert. Die Zahlen in Klammern zeigen jeweils an, wieviele Schritte sich in dieser Gruppe befinden.
 
Hauptfenster: Backups
Bei jeder Benutzung eines Wizard legt das CWF eine Sicherheitskopie Ihrer aktuellen Konfiguration an, solange es sich um eine Konfiguration über Konfigurationsdateien handelt. Sollte Ihnen bei der Konfiguration der Applikation ein Fehler unterlaufen können Sie jederzeit eine alte Version wiederherstellen. Beim wiederherstellen wird die aktuelle Version auch wieder als Backup gespeichert. Sind Backups vorhanden, zeigt Ihnen das Fenster das Datum und die Zeit der letzten Änderung an, sowie die Dateigrösse in Bytes. Sie können dann jede der angezeigten Konfigurationen wieder herstellen oder unwiderruflich löschen.
