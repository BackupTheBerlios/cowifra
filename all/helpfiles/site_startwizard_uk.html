The wizard step selection window
This window shows you all steps of the current wizard. Naturally you start a wizard by clicking the wizard start button. But sometimes you would like to change only one step. In that case you can reach this step by using this menue. The checked icon shows the position of the current step in all steps of the wizard. You can hide/unhide this window by clicking the small button in the headline.
 
The wizard actions window
In this window you will find all actions that should take place after changing something in any step. In most cases you can run these actions by clicking the entry "Run all actions". If you know what you're doing, you can run single actions. This window appears only, if there are such actions. You can hide/unhide this window by clicking the small button in the headline.
 
Main content: Header
The first section of the wizard startpage gives you several information about the current wizard file and the author. You should verify the right application version, to avoid wrong results. If you find any mistakes in a wizard file you can contact the author, his email adress should also be here.
 
Main content: Wizard start button
Naturally you should start a wizard by using this button. The number in brackets is the step counter, in this sample you have to complete 44 steps.
 
Main content: Wizard start by group
It depends on the author of the wizard file if this groups options are available in a wizard. There are two optional group options.
Group by user level - take a look at the defined user levels and choose the right one for your needs.
Group by theme - take a look at the defined themes and choose the right one for your needs.
In this sample there are to diffrent user levels and three themes defined. The numbers in brackets represents the number of steps in each group.
 
Main content: Backup files
Each time you use the configuration wizard framework (cwf) to change configuration files of an application, the cwf create a backup file of the orginal configuration. If something goes wrong or if you like to restore an old configuration you're here at the right place. The window show you the date, time and size of each backup file. It also gives you the chance to restore or delete a backup file. If you restore an old configuration your actual configuration will be saved in a backup file.
