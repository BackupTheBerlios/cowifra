Wizard Dateien (*.cwf)
Jeder Wizard für eine Anwendung ist in einer Wizard Datei gespeichert, die sich im "wizards" Unterverzeichnis befinden. Die Informationen in den Wizard Dateien werden im vom W3C freigegebenen XML-Format zur Verfügung gestellt. Die Syntax dieser XML-Dateien ist in der "document type definition" (dtd) definiert, die sich ebenfalls im wizards Unterverzeichnis mit dem Namen wizard.dtd befindet.

Wenn Sie eine Linux bzw. Unix Anwendung mit dem CWF einrichten möchten, benötigen Sie zunächst einmal die entsprechende Wizard Datei. Auf die, zur Anwendung passende, Versionsnummer ist dabei zu achten. Auf der Startseite des CWF werden Ihnen alle in Ihrem System zur Verfügung stehenden Wizards angezeigt, wenn der gewüschte Wizard nicht angezeigt wird, müssen Sie diesen von aus dem World Wide Web (WWW) herunterladen, z.B. von http://cwf.berlios.de (Adresse zur Zeit noch nicht verfügbar).

Informationen zum erstellen eigener Wizards finden Sie in der "Anleitung" (Diplomarbeit) zum CWF.
